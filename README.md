
# Blowfish

## Introduction

:blowfish: A Blog

`Not compatible with low version browsers`

`node version>=5.9.0`

## dependencies
- mognodb
- redis
- nodemon
- pm2

```bash
$ mongod &
$ redis &
```

## development

```bash
$ npm run dev:server
$ npm run dev:client
```

## production

```bash
$ npm run start
```
